# Quiz-Telegram-Bot
## Список команд
* **/start** - Запуск бота
* **/quiz** - Начать квиз
* **/result** - Показать результат
## Структура проекта
* **main.py** - тело бота
* **db.py** - запросы к БД
* **questions** - список вопросов
* **settings.py** - настройки
* **input** - bot token
